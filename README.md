# Student Exam Score Prediction

## Mini Project

This project predicts student exam scores using Machine Learning.

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Streamlit

## Project Structure

src/
- load_data.py
- data_cleaning.py
- visualization.py
- model_training.py
- app.py

## How to Run

Load Dataset

```bash
py src/load_data.py
```

Data Cleaning

```bash
py src/data_cleaning.py
```

Visualization

```bash
py src/visualization.py
```

Train Model

```bash
py src/model_training.py
```

Run Streamlit App

```bash
streamlit run src/app.py
```
